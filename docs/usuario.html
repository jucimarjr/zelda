3.2. Tabela de Usuário

Coluna Tipo de Dado Descrição
usuario_id int No atributo usuario_id, ficam armazenados os dados referentes a identificação de cada usuário, onde esse dado é uma chave primária e irá guardar uma identificação de cada usuário cadastrado no banco de dados.
usuario_login varchar No atributo usuario_login, ficam armazenados os dados referentes ao login que o usuário cadastrou.
usuario_senha varchar No atributo usuario_senha, fica armazenado a senha que o usuário cadastrou.
usuario_logado int No atributo usuario_logado, irá definir se o usuário está logado, definindo como 1 caso esteja logado ou 0 caso contrário.
usuario_admin int No atributo usuario_admin, irá guardar se o usuário é administrador ou não.
